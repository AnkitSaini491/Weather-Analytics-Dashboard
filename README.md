# Weather-Analytics-Dashboard
A professional Weather Analytics Dashboard built using Python, Pandas, and Matplotlib. It provides weather trend analysis through interactive visualizations, KPI cards, and multiple charts for temperature, humidity, wind speed, rainfall, and weather distribution. Designed with a clean dark theme to demonstrate data analysis and dashboardills.

# Weather Analytics Dashboard

A professional weather analytics dashboard built using Python, Pandas, and Matplotlib.

## Features

- Temperature Trend
- Humidity Analysis
- Wind Speed Analysis
- Rainfall Analysis
- Weather Distribution
- KPI Cards
- Dark Theme Dashboard

## Technologies

- Python
- Pandas
- Matplotlib

## Run

pip install -r requirements.txt

python weather_dashboard.py
